# Minecraft Sencillo (Web)

Un mini juego inspirado en Minecraft: mundo 2D con bloques, selección de materiales y colocación/eliminación con clic.

## Cómo jugar

1. Abre `index.html` en tu navegador.
2. Selecciona un bloque en la barra inferior.
3. Clic izquierdo: colocar bloque.
4. Clic derecho: eliminar bloque.
5. Usa la rueda del ratón para acercar/alejar.

## Controles

- `1-6`: seleccionar bloque rápido.
- `Espacio`: alterna modo colocar/eliminar.
- `R`: regenerar el mundo.
